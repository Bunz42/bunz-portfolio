## My Favorite Features
* **One-Page Layout:** Smooth scrolling between sections.
* **Intro Animation:** Cool intro animation when visiting the website.
* **Glass Morphism Styling:** Containers are glass morphism style.
* **3D About Me Container:** Cool 3D effect on the about me container.
* **Container Borders:** Red pulsating container borders.
* **Contact Form:** Links to my relevant socials.
